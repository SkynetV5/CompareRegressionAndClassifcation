# CompareRegressionAndClassifcation

## Requirements
- Python >=3.12
- Pandas ==2.2.3
- Matplotlib == 3.10.1
- Tinker
- Numpy ==2.1.2

## Clone repository

 ```git clone https://github.com/SkynetV5/CompareRegressionAndClassifcation.git```

## Create and Active virtual env

Unix/MacOS
```python3 -m venv venv```
```source venv/bin/activate```

Windows
```python -m venv venv```
```venv\Scripts\activate```

## Start 

```python main.py```
